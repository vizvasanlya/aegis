# Rebrand Strix → Aegis

## Goal
Rename the entire codebase from "Strix" to "Aegis" across all layers: Python package, CLI, Docker image, docs, scripts, frontend, and config. The copyright holder in LICENSE must remain "OmniSecure Inc." per Apache 2.0 requirements.

## Scope Summary
~190+ individual changes across ~45+ files. The Python package directory rename is the highest-risk change.
