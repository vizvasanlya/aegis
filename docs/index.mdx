## Use Cases

- **Application Security Testing** — Detect and validate critical vulnerabilities
- **Rapid Penetration Testing** — Get tests done in hours, not weeks
- **Bug Bounty Automation** — Automate research and generate PoCs
- **CI/CD Integration** — Block vulnerabilities before production

## Key Capabilities

- **Full hacker toolkit** — Browser automation, HTTP proxy, terminal, Python runtime
- **Real validation** — PoCs, not false positives
- **OWASP checklist** — Mandatory 8-category testing coverage
- **Multi-agent orchestration** — Specialized agents collaborate on complex targets

## Security Tools

Aegis agents come equipped with a comprehensive toolkit:

| Tool | Purpose |
|------|---------|
| HTTP Proxy | Full request/response manipulation and analysis |
| Browser Automation | Multi-tab browser for XSS, CSRF, auth flow testing |
| Terminal | Interactive shells for command execution |
| Python Runtime | Custom exploit development and validation |

## Mandatory Testing Categories

Every scan completes all 8 categories:

1. **Authentication & Session** — Login, JWT, OAuth, MFA
2. **Access Control** — IDOR, privilege escalation
3. **Injection** — SQLi, XSS, NoSQLi, SSTI, XXE
4. **Server-Side** — SSRF, path traversal, deserialization
5. **Client-Side** — DOM XSS, CSRF, clickjacking
6. **Configuration** — CORS, CSP, security headers
7. **Business Logic** — Race conditions, workflow bypass
8. **API Security** — REST/GraphQL auth, data exposure

## Get Started

```bash
# Clone and install
git clone https://github.com/vizvasanlya/aegis.git
cd aegis && uv sync

# Configure
export AEGIS_LLM="openai/gpt-4o"
export LLM_API_KEY="your-api-key"

# Run your first scan
aegis --target https://example.com
```
