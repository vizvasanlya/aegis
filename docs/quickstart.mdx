## Prerequisites

- Docker (running)
- An LLM API key from any [supported provider](/llm-providers/overview) (OpenAI, Anthropic, Google, etc.)

## Installation

```bash
# Clone the repository
git clone https://github.com/vizvasanlya/aegis.git
cd aegis

# Install dependencies
uv sync
```

## Configuration

Set your LLM provider:

```bash
export AEGIS_LLM="openai/gpt-4o"
export LLM_API_KEY="your-api-key"
```

<Tip>
For best results, use `openai/gpt-4o`, `anthropic/claude-sonnet-4-6`, or `opencode/deepseek-v4`.
</Tip>

## Run Your First Scan

```bash
aegis --target ./your-app
```

<Note>
First run pulls the Docker sandbox image automatically. Results are saved to `aegis_runs/<run-name>`.
</Note>

## Target Types

Aegis supports multiple target types:

```bash
# Local codebase
aegis --target ./app-directory

# GitHub repository
aegis --target https://github.com/org/repo

# Web application
aegis --target https://your-app.com

# Multiple targets
aegis -t ./app -t https://api.your-app.com
```

## Scan Modes

```bash
# Quick scan (10-15 min)
aegis --target ./app --scan-mode quick

# Standard scan (20-30 min)
aegis --target ./app --scan-mode standard

# Deep scan (30-60 min)
aegis --target ./app --scan-mode deep
```

## Headless Mode

Run without interactive UI:

```bash
aegis -n --target https://your-app.com
```
