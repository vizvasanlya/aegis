[OpenCode Zen](https://opencode.ai/docs/zen/) is an AI gateway that provides access to tested and verified models from multiple providers through a single API. It curates the best model/provider combinations for coding agents.

## Setup

```bash
export STRIX_LLM="opencode/gpt-5.5"
export OPENCODE_API_KEY="your-zen-api-key"
```

That's it — the base URL is auto-configured when you use the `opencode/` prefix.

## Get API Key

1. Go to [opencode.ai](https://opencode.ai)
2. Sign in and navigate to **Settings → API Keys**
3. Create a new API key

## Custom Base URL (Advanced)

If you need to override the base URL (e.g., for a proxy):

```bash
export STRIX_LLM="opencode/gpt-5.4"
export OPENCODE_API_KEY="your-zen-api-key"
export LLM_API_BASE="https://custom-proxy.example.com/zen/v1"
```

## Example

```bash
# Run a penetration test with a free model
export STRIX_LLM="opencode/deepseek-v4-flash-free"
export OPENCODE_API_KEY="your-zen-api-key"
strix --target https://example.com --scan-mode quick

# Use a high-performance model for deep testing
export STRIX_LLM="opencode/claude-opus-4-8"
strix --target ./app --scan-mode deep
```
